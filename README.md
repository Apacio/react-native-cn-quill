# react-native-cn-quill

react-native richtext editor with quilljs

## Installation

```sh
npm install react-native-cn-quill
```

## Usage

```js
import CnQuill from "react-native-cn-quill";

// ...

const result = await CnQuill.multiply(3, 7);
```
